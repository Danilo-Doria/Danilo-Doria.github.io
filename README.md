## 📖 Description

The main goal of this project is to introduce me into the world of web development and improve my **HTML** layout skills and my ability to create responsive designs using **CSS**, as well as to gain an introduction to **JavaScript**.

The objective is to create a responsive website that includes:
- Personal information
- Featured projects
- A contact form or contact information

---

## ✨ Features

- Responsive design 📱💻
- Navigation bar
- Animations with CSS
- Layout with Flexbox and Grid
- Basic interactivity with JavaScript
