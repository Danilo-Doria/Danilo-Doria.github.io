Sobre esta galería

Esta galería muestra diferentes tipos de mascotas, principalmente perros, organizados en un diseño moderno con Grid y tarjetas visuales.

Cada tarjeta incluye una imagen, nombre y descripción breve para practicar maquetación web y diseño responsive.
